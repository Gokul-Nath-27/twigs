# Table

### Import

```jsx
import { Table, Tbody, Td, Th, Thead, Tr } from "@sparrowengg/twigs-react";
```

All these components are a wrapper around the html elements `table`, `tbody`, `td`, `th`, `thead`, `tr` respectively.

### Usage 
```jsx live
<Table>
    <Thead>
      <Th>Name</Th>
      <Th>Company</Th>
      <Th>Department</Th>
      <Th>Joined on</Th>
    </Thead>
    <Tbody>
      <Tr>
        <Td>Justin</Td>
        <Td>SurveySparrow</Td>
        <Td>Engineering</Td>
        <Td>Sep, 2020</Td>
      </Tr>
      <Tr>
        <Td>Maneesh</Td>
        <Td>SurveySparrow</Td>
        <Td>Engineering</Td>
        <Td>Sep, 2020</Td>
      </Tr>
    </Tbody>
  </Table>
```
