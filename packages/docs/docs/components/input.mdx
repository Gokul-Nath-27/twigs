## Props
| Property  | Description                            | Type            | Default |
|-----------|----------------------------------------|-----------------|---------|
| variant   | style of input                         | default, filled | default |
| onChange  |                                        | function        |         |
| size      | input size                             | sm,md,lg,xl     | md      |
| disabled  |                                        | boolean         | false   |
| iconLeft  | Append icon on the left side of input  | JSX Element     |         |
| iconRight | Append icon on the right side of input | JSX Element     |         |
| css       | Custom styles for input                | object          |         |
