# Line Loader

### Import

```jsx
import { LineLoader } from "@sparrowengg/twigs-react";
```

### Usage 
```jsx live
<Box css={{ background:'$secondary100', padding:'$6' }}>
  <LineLoader size="xl" />
</Box>
```

### Props
| Property       | Description                                | Type                    | Default |
|----------------|--------------------------------------------|-------------------------|---------|
| size           | size of the loader                         | `sm` `md` `lg` `xl`     | sm      |
