## ThemeProvider
Twigs come with a `ThemeProvider` which is completely customizable. With the help of `ThemeProvider`, you can modify attributes such as colors, font sizes, spacing, etc. 

<ol>
    <li><a href="#colors">Colors</a></li>
    <li><a href="#fontSize">Font Size</a></li>
    <li><a href="#fontWeights">Font Weights</a></li>
    <li><a href="#borderWidths">Border Widths</a></li>
    <li><a href="#borderRadius">Border Radius</a></li>
</ol>


Basic usage of `ThemeProvider`
```jsx
import { ThemeProvider } from '@sparrowengg/twigs-react';

const Index = () => (
<ThemeProvider>
    <App /> //your app entry point
</ThemeProvider>
)

```


Optionally you can pass a `theme` object to `ThemeProvider` to modify the default theme. 

```jsx
<ThemeProvider
    theme={{
    fonts: {
        body: "'DM sans', sans-serif",
    },
    }}
>
    <App /> //your app entry point
</ThemeProvider>
```
